# Favoriten Manager ⭐️

Ein moderner Favoriten-Manager mit Todo-style Layout, Kategorien und Drag & Drop.

## Features
- 🚀 **Lokale Entwicklung:** Schneller Node.js Server mit lokaler Speicherung.
- 📁 **Kategorien:** Erstelle beliebig viele Spalten für deine Projekte.
- 🖱️ **Drag & Drop:** Sortiere deine Favoriten einfach per Maus.
- 📥 **Import:** Importiere deine Browser-Lesezeichen (HTML-Export).

## Lokale Installation & Start
1. Stelle sicher, dass [Node.js](https://nodejs.org/) installiert ist.
2. Lade diesen Ordner herunter.
3. Klicke doppelt auf die Datei **`start_app.bat`**.
4. Die App öffnet sich automatisch unter `http://localhost:3000`.

## GitHub-Vorbereitung
Die App ist so vorbereitet, dass sie in Zukunft auch direkt über GitHub Pages laufen kann. 
Aktuell nutzt sie zur Entwicklung den lokalen Node.js Server.
